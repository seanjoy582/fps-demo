# 军事体验 FPS Demo

一个基于 Three.js ES Modules 的浏览器第一人称射击 Demo。

## 本地运行

```powershell
node dev-server.js
```

然后打开：

```text
http://localhost:5173/index.html
```

## 操作

- `W A S D` 移动
- 鼠标控制视角
- 左键射击
- 右键瞄准
- `R` 换弹
- `1` AK47
- `2` M4A1
- `3` M249
- `4` 手枪

## 验证

```powershell
node smoke-test.js
```
