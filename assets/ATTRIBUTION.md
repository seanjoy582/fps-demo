# Asset attribution

This project currently uses project-generated SVG textures, sprites, and weapon silhouettes created specifically for this local FPS demo.

External references searched for future replacement candidates:
- Wikimedia Commons for openly licensed firearm reference imagery.
- OpenGameArt and Freesound for openly licensed gunshot/reload sounds.
- ambientCG and Poly Haven for CC0 ground/concrete/metal texture references.

No commercial game assets are included.
